# docs/evals - agent notes

## Scope

These rules apply to eval scenario documentation and rubrics.

## Rules

- Keep scenarios adversarial and behavior-focused, not tailored to current
  implementation wording.
- When a rubric changes, check the corresponding tests or eval runner still
  measure the same behavior.
- Do not weaken a scenario to make a current model pass.

## Verification

- Run the nearest eval or targeted test when scenario semantics change.
